# Package index

## All functions

- [`autoreweight()`](https://emvolz.github.io/cod/reference/autoreweight.md)
  : Automatically reweight sample units that may be oversampled
- [`chindex()`](https://emvolz.github.io/cod/reference/chindex.md) :
  Computes the CH index for selecting clustering thresholds
- [`chindices()`](https://emvolz.github.io/cod/reference/chindices.md) :
  Compute CH index over a range of clustering thresholds
- [`codbinomial()`](https://emvolz.github.io/cod/reference/codbinomial.md)
  : Fit a genealogical placement GMRF model using maximum likelihood
- [`codls()`](https://emvolz.github.io/cod/reference/codls.md) : Fit a
  COD GMRF model using weighted least squares
- [`codml()`](https://emvolz.github.io/cod/reference/codml.md) : Fit a
  genealogical placement GMRF model using maximum likelihood
- [`coef.gpgmrf()`](https://emvolz.github.io/cod/reference/coef.gpgmrf.md)
  : A Capitalized Title (ideally limited to 65 characters)
- [`computeclusters()`](https://emvolz.github.io/cod/reference/computeclusters.md)
  : Compute phylogenetic clusters by cutting tree at branches with large
  changes in coalescent odds
- [`fitgpgmrf()`](https://emvolz.github.io/cod/reference/fitgpgmrf.md) :
  A Capitalized Title (ideally limited to 65 characters)
- [`lbi()`](https://emvolz.github.io/cod/reference/lbi.md) : Compute the
  local branching index (LBI)
- [`neffextant()`](https://emvolz.github.io/cod/reference/neffextant.md)
  : Effective number of extant lineages
- [`optcodsmooth()`](https://emvolz.github.io/cod/reference/optcodsmooth.md)
  : Evaluate the loss function of the cod model across a range of tau
  (precision parameter) values
- [`optsmooth()`](https://emvolz.github.io/cod/reference/optsmooth.md) :
  Optimise smoothing parameter for a given tree and branch statistic
- [`plot.gpgmrf()`](https://emvolz.github.io/cod/reference/plot.gpgmrf.md)
  : A Capitalized Title (ideally limited to 65 characters)
- [`plotclusters()`](https://emvolz.github.io/cod/reference/plotclusters.md)
  : Plots a fit from \`codls\` and cluster assigment from
  \`computeclusters\`
- [`print.gpgmrf()`](https://emvolz.github.io/cod/reference/print.gpgmrf.md)
  : A Capitalized Title (ideally limited to 65 characters)
- [`rmsclo()`](https://emvolz.github.io/cod/reference/rmsclo.md) : Root
  mean square coalescent log odds
- [`tauprofile()`](https://emvolz.github.io/cod/reference/tauprofile.md)
  : Evaluate the loss function of the cod model across a range of tau
  (precision parameter) values
