# Compute the local branching index (LBI)

Compute the local branching index (LBI)

## Usage

``` r
lbi(tr, logtau)
```

## Arguments

- tr:

  An ape::phylo

- logtau:

  Smoothing parameter

## Value

Vector of local branching index values for every node in the input tree

## References

Neher, Richard A., Colin A. Russell, and Boris I. Shraiman. Elife 3
(2014): e03568.
